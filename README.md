# Конвертер валют

Простой и удобный конвертер валют на Python, использующий открытый API для получения актуальных курсов валют.

## Возможности

- ✅ Конвертация между любыми валютами
- ✅ Получение актуальных курсов валют в реальном времени
- ✅ Интерактивный интерфейс командной строки
- ✅ Кэширование данных для оптимизации запросов
- ✅ Поддержка более 150 валют
- ✅ Автоматическое использование данных ЦБ РФ для курсов с рублем

## Установка

1. Убедитесь, что у вас установлен Python 3.7 или выше

2. Установите необходимые зависимости:
```bash
pip install -r requirements.txt
```

## Использование

### Интерактивный режим

Запустите программу:
```bash
python currency_converter.py
```

Затем следуйте инструкциям в меню:
- **1** - Конвертировать валюту
- **2** - Узнать курс обмена
- **3** - Показать список популярных валют
- **4** - Обновить курсы валют (очистить кэш)
- **5** - Выход

### Программное использование

Вы можете использовать класс `CurrencyConverter` в своем коде:

```python
from currency_converter import CurrencyConverter

# Создание экземпляра конвертера
converter = CurrencyConverter()

# Конвертация валюты
result = converter.convert(100, "USD", "EUR")
print(f"100 USD = {result} EUR")

# Получение курса обмена
rate = converter.get_rate("USD", "RUB")
print(f"1 USD = {rate} RUB")

# Получение списка доступных валют
currencies = converter.get_available_currencies()
print(f"Доступно валют: {len(currencies)}")
```

## Поддерживаемые валюты

Программа поддерживает все основные валюты мира, включая:
- USD (Доллар США)
- EUR (Евро)
- GBP (Фунт стерлингов)
- JPY (Японская йена)
- CNY (Китайский юань)
- RUB (Российский рубль) - с актуальными данными от ЦБ РФ
- И многие другие...

Полный список доступных валют можно получить через метод `get_available_currencies()`.

## API

Используется бесплатный API [exchangerate-api.com](https://www.exchangerate-api.com/) для получения курсов валют.

Для пар с участием российского рубля (RUB) дополнительно используется API [Центробанка России](https://www.cbr-xml-daily.ru/) для получения наиболее актуальных курсов.

## Особенности

- **Кэширование**: Данные кэшируются на 5 минут для оптимизации запросов
- **Актуальные курсы RUB**: Автоматическое использование данных ЦБ РФ для всех пар с рублем
- **Обработка ошибок**: Программа корректно обрабатывает ошибки сети и API
